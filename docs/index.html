KittenTTS Web

Tiny, fast text-to-speech that runs on CPU. No GPU needed.

Run locally

git clone https://github.com/DipFlip/KittenTTSWeb.git
cd KittenTTSWeb
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m uvicorn webui:app --host 127.0.0.1 --port 8000

Then open http://localhost:8000.
